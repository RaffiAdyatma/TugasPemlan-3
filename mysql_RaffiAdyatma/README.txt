APLIKASI GUI JAVA UNTUK MELIHAT DAN MENGEDIT TABEL DATABASE

Jalankan file Main.java di dalam src

Cara pakai;
ketikkan nama database dan tekan tombol 'connect', akan mengkoneksikan ke database yang dituju
ketik nama dari tabel dan tekan pilih untuk melihat isi tabel dan set tabel

ketikkan query di bagian bawah dan tekan 'run' untuk menjalankan

untuk program ini, memiliki query spesial "ADD ()"
sama dengan "INSERT", ADD menambahkan tanpa harus menulis "INSERT INTO TABEL (KEY1, KEY2, ...) VALUES ..."
hanya perlu menuliskan "ADD (KEY1, KEY2, KEY3, ...), (KEY1, KEY2, KEY3, ...)"
pastikan tabel sudah di set, dan syntax sama dengan bagian 'VALUES' untuk mySQL
